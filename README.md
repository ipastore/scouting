## Installation

1) Clone Repo

2) Make environment
If you are familiar with conda:

conda env create -f environment.yml
conda activate myenv

If you are familir with pip: 

python3 -m venv env
source env/bin/activate
pip install -r requirements.txt

## Run
streamlit run reports_app.py
